Instructions

Welcome to the graphical web crawler!

The graphical web crawler begins at a designated starting webpage and navigates to linked pages. The results of the web crawler are displayed in a graphical format. To begin, fill out the form and click submit.

Starting webpage: Provide the URL for the webpage to start crawling from.

Search method: Choose either depth-first or breadth-first search.

Numeric limit: For a depth-first search, the numeric limit is 100 pages. For a breadth-first search, the numeric limit is 3 levels. (Due to the large number of pages visited by a breadth-first search, we suggest limiting the numeric limit to 1.)

Keyword: Providing a keyword is optional. If the web crawler visits a webpage that contains the keyword, the traversal will stop. The webpage that contained the keyword will be highlighted in the graph.

Time limit: Please be advised that the crawler may take some time to visit the requested number of pages or levels, especially when performing a breadth-first search. To this end, we imposed a 1 minute limit on all crawler searches. If a search does not complete in this time, you will be notified and can perform another search.

Graph manipulation: After the graph is displayed, it can be manipulated: drag to pan, scoll to zoom, and click on a node to see more information about that page.

Previous Searches: Your 10 most recent searches are stored locally using cookies. You can view these by clicking the "View Previous Searches" link.
